## Getting Started

### First Time Users

1. **Launch Application**: Double-click the CampusConnect application or run from command line
2. **Login Screen**: You'll see the main login interface
3. **Create Account**: Click "Register" to create a new account
4. **Choose Role**: Select your role (Student, Club Member, or Admin)

### Account Types

- **Student**: Browse events, join clubs, register for activities
- **Club Member**: Create events, manage club activities, organize activities
- **Administrator**: Approve clubs/events, manage users, system oversight

## Student Guide 🎓

### Registration Process

1. **Personal Information**
   - Enter your full name and email address
   - Create a secure password
   - Confirm your password

2. **Academic Details**
   - Select your department from the dropdown
   - Choose your current semester (1-8)
   - Add your interests (comma-separated)

3. **Account Verification**
   - Your account will be created immediately
   - You can log in with your credentials

### Dashboard Overview

Upon login, you'll see your Student Dashboard with:
- **Upcoming Events**: Events you've registered for
- **Available Events**: Browse new events to join
- **My Clubs**: Clubs you're a member of
- **Quick Actions**: Register for events, join clubs, view profile

### Browsing Events

1. **View All Events**: Click "View Events" from dashboard
2. **Event Details**: Each event shows:
   - Event name and description
   - Date, time, and venue
   - Club organizing the event
   - Available spots remaining
   - Ticket price (if applicable)

3. **Filtering Events**
   - Use search to find specific events
   - Filter by date range
   - Filter by club or category

### Registering for Events

1. **Select Event**: Click on an event you're interested in
2. **Check Details**: Review event information carefully
3. **Register**: Click "Register" button
4. **Payment** (if required):
   - Enter payment details
   - Confirm payment amount
   - Complete transaction

5. **Confirmation**: You'll receive confirmation of registration

### Joining Clubs

1. **Browse Clubs**: Navigate to "View Clubs" section
2. **Club Information**: View club details:
   - Club name and category
   - Description and activities
   - Current member count
   - Club status (Active/Pending)

3. **Join Request**: Click "Join Club" to send a membership request
4. **Approval**: Wait for club administrators to approve your request

### Managing Your Profile

1. **View Profile**: Click your name or "Profile" in the menu
2. **Personal Information**: Update name, email, or password
3. **Academic Information**: Modify department, semester, or interests
4. **Event History**: View past events you've attended

### My Participations

Track your campus involvement:
- **Registered Events**: Upcoming events you're registered for
- **Past Events**: History of events you've attended
- **Club Memberships**: List of clubs you belong to
- **Payment History**: Record of event payments

## Club Member Guide 🏛️

### Club Creation Process

1. **Request New Club**
   - Navigate to "Create Club" section
   - Fill in club details:
     - Club name (must be unique)
     - Category (Academic, Cultural, Sports, etc.)
     - Description and objectives
     - Initial member information

2. **Approval Process**
   - Submit club creation request
   - Wait for administrator approval
   - You'll be notified of approval status

### Club Dashboard

Once your club is approved:
- **Club Overview**: Member count, status, recent activity
- **Event Management**: Create and manage club events
- **Member Management**: View and manage club members
- **Requests**: Handle membership requests from students

### Creating Events

1. **Event Planning**
   - Click "Create Event" from club dashboard
   - Enter event details:
     - Event name and description
     - Date and time (start/end)
     - Venue information
     - Maximum participants
     - Ticket pricing

2. **Event Configuration**
   - Set registration limits
   - Enable/disable discounts
   - Add special requirements
   - Upload event materials

3. **Submit for Approval**
   - Review all event details
   - Submit to administrators for approval
   - Monitor approval status

### Managing Events

1. **View Created Events**: See all events your club has organized
2. **Event Status Tracking**:
   - **Pending**: Waiting for admin approval
   - **Approved**: Open for student registration
   - **Completed**: Event has finished
   - **Cancelled**: Event was cancelled

3. **Registration Management**:
   - View registered participants
   - Download participant lists
   - Send communications to registrants

### Member Management

1. **View Members**: See all club members and their roles
2. **Membership Requests**: 
   - Review pending requests from students
   - Approve or reject membership applications
   - Send welcome messages to new members

3. **Role Management**:
   - Assign member roles (President, Member)
   - Update member information
   - Remove inactive members

### Club Profile Management

1. **Club Information**: Update club description, category, or contact details
2. **Activity History**: View past events and member activities
3. **Statistics**: Monitor club growth and engagement metrics

## Troubleshooting

### Common Issues

#### Login Problems
**Issue**: Cannot log in to account
**Solutions**:
- Verify username and password
- Check for caps lock
- Contact administrator for password reset

#### Registration Issues
**Issue**: Cannot register for events
**Solutions**:
- Check if event is full
- Verify you're not already registered
- Ensure payment method is valid

#### Club Access Problems
**Issue**: Cannot access club features
**Solutions**:
- Verify your club membership status
- Check if club is approved
- Contact club administrators

### Getting Help

1. **Check Documentation**: Review this user guide and FAQ
2. **Contact Support**: Reach out to system administrators
3. **Report Bugs**: Use the feedback system to report issues
4. **Community Help**: Ask other users in club forums

### Performance Tips

- **Keep Browser Updated**: Use latest version for best performance
- **Clear Cache**: Clear browser cache if experiencing slow loading
- **Stable Internet**: Ensure good internet connection for smooth operation
- **Regular Updates**: Keep the application updated

### Error Messages

Common error messages and their meanings:

- **"Event Full"**: Maximum participants reached
- **"Invalid Credentials"**: Incorrect username or password
- **"Access Denied"**: Insufficient permissions for requested action
- **"Payment Failed"**: Issue with payment processing
