CampusConnect - Campus Event Management System

CampusConnect is a comprehensive campus event management system designed to facilitate event organization, club management, and student participation in campus activities. The system provides role-based functionality for students, club administrators, and system administrators.

Key Features

  • Event Management: Create, approve, and manage campus events
  • Club Administration: Register clubs, manage memberships, and organize activities
  • Student Participation: Browse events, register for activities, and track participation history
  • Administrative Oversight: Approve clubs and events, generate reports, and manage system-wide settings

System Architecture

The application follows a layered architecture pattern with the following main components:

  • Model: Entity classes representing core business objects (Club, Event, User, Student, etc.)
  • DAO (Data Access Objects): Database interaction layer with CRUD operations
  • Service: Business logic layer containing application-specific operations
  • Controller: JavaFX UI controllers for user interface management
  • Utils: Utility classes for common operations and validation
  • DataStructures: Custom data structures optimized for specific use cases
  • ExceptionHandler: Custom exceptions for better error handling

Technology Stack

  • Java 17: Core programming language
  • JavaFX: User interface framework
  • PostgreSQL: Database management system
  • JDBC: Database connectivity

User Roles

  • Students: Can browse events, join clubs, register for events, and view participation history
  • Club Administrators: Can create and manage clubs, organize events, and manage club membership
  • System Administrators: Can approve clubs and events, generate reports, and manage system settings
